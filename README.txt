== Installation ==

You can download and install  WPDEV SLIDER plugin using the built in WordPress plugin installer. If you download WPDEV SLIDER manually, make sure it is uploaded to "/wp-content/plugins/wpdev-slider/".

Activate WPDEV SLIDER in the "Plugins" admin panel using the "Activate" link.

You'll then see WPDEV SLIDER dashboard from which you can enable or disable the button as per your need.
